Семестровая работа (работа в командах из 2-3 человек):

Создать веб-приложение с дальнейшим расширением на использование веб-сервисов. 
Первый блок по приложению (семестровка):

1. Описать модели для приложения, создать БД.
2. Создать аутентификацию(middleware).
3. Приложение должно давать возможность добавлять данные через формы.
Разграничить доступ к данным:
4. Приложение должно давать возможность просматривать/редактировать/удалять список своих объектов.
5. Приложение должно давать возможность просматривать список всех объектов.
6. В пунктах 4, 5 должна быть возможность фильтрации/сортировки данных.
7. Должна быть реализована работа с файлами, с возможностью их изменения и скачивания, т.е. модель данных должна предполагать хранение или формирование неких файлов, который будут храниться на жестком диске (в БД будет храниться только путь до файла).
