 Последовательные натуральные числа от a до b выписали одно за другим без пропусков, при этом образовалось число, которое без остатка делится на натуральное c. По данным a и c указать наименьшее значение b.

Входные данные

Два числа a и c (a, c < 10000, a ≤ b).

Выходные данные

Вывести значение b.
