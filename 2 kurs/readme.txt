Задания следует выполнять последовательно по возрастанию номеров.
После выполнения каждого отдельного задания, следует закоммитить его с комментарием 
"[done] exercise #"
где # - это номер задания, которое вы выполнили.
Существует следующая система дедлайнов (будет дополняться):
Дата 			Задание
25.02.2020		exercise.txt
25.02.2020		exercise1.txt
